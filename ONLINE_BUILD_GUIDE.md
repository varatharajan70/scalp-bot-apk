# 🚀 Build APK Using Online Builder (5 Minutes!)

**No setup, no commands, just upload and download!**

---

## 📋 **What You Need**

Two files from this folder:
1. ✅ `main.py` - Main app (already created)
2. ✅ `buildozer_online.spec` - Build config (already created)

---

## 🌐 **Option A: p3x.io APK Builder (Recommended)**

### Step 1: Go to Website
**URL:** https://p3x.io/downloads/apk-builder/index.html

### Step 2: Upload Files
1. Click **"Add File"** or drag files
2. Upload **`main.py`** (the main app)
3. Upload **`buildozer_online.spec`** (config)

### Step 3: Configure (Optional)
- **App Name:** SCALP BOT TERMINAL
- **Package:** org.scalpbot.scalpbot
- **Version:** 1.0

### Step 4: Build
- Click **BUILD APK**
- Wait 5-10 minutes
- See progress bar

### Step 5: Download
- Download APK when ready
- File: `scalpbot-1.0-debug.apk`

---

## 🌐 **Option B: Buildozer Online**

**URL:** https://buildozer-online.com/

**Steps:**
1. Drag & drop `main.py`
2. Select "Android"
3. Click "Build"
4. Download when ready

---

## 🌐 **Option C: Kivy Cloud Builder**

**URL:** https://kivy.org/doc/stable/guide/packaging-android.html

---

## 📱 **After Download**

### Transfer to Phone
```
1. Download APK on computer
2. Connect phone via USB cable
3. Drag APK to phone storage
   (or use "Send to device" in file manager)
```

### Install on Phone
```
1. Open File Manager on phone
2. Find downloaded APK
3. Tap APK file
4. Click "INSTALL"
5. Wait for installation
6. Click "OPEN"
```

### First Run
```
1. App opens with terminal
2. See: "🤖 SCALP BOT TERMINAL v1.0"
3. Click ▶️ START button
4. Bot starts running
5. See live output in terminal
6. Click ⏹️ STOP to stop
```

## 🔧 **Troubleshooting**

### "APK file too large"
- Normal for Kivy apps (60-80 MB)
- Phone storage: 150+ MB needed

### "Installation failed"
- Check: Android version 5.0+
- Try: Uninstall old version first
- Check: Enough storage space

### "App crashes on start"
- Try: Clear app cache (Settings → Apps)
- Try: Reinstall APK
- Check: Permissions granted

### "Status stuck at STOPPED"
- This is normal for demo
- Click START to activate
